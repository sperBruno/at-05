##Git Comands
---
1. Configure user
> git config --user.name "Bruno barrio"
> git config --user.email "blbarrios@outlook.com"

2. Git status
> git status

3. Initialize git repository
> git init

4. See commits
> git log

5. Move to a branch
> git checkout <branch name>

6. Clone a repository
> git clone <url>

7. Look logs
> git --oneline
